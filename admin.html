📌 Panduan Menyimpan Perubahan (PENTING!)

Karena website ini adalah Static Website, perubahan yang Anda lakukan di halaman ini tidak akan otomatis tersimpan di server. Ikuti langkah-langkah berikut:

  1. Lakukan penambahan/perubahan data produk atau pengaturan WA pada form di bawah.
  2. Jika sudah selesai, klik tombol "Download products.csv" dan/atau "Download config.js".
  3. Buka repository GitHub Anda, lalu lakukan Replace (timpa) file lama yang ada di folder data/products.csv dan config/config.js dengan file yang baru saja didownload.
  4. Untuk Gambar: Anda wajib mengupload file gambar secara manual ke folder assets/categories/ atau assets/products/ di GitHub.
